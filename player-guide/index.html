Общий регламент

Блок ниже - шаблон. Оператор турнира заполняет название, режим и контакты перед событием (см. также docs/TOURNAMENT-REGULATIONS.md в репозитории).

  • Регистрация на турнир используя SteamID
  • Игра на серверах турнира от своего Steam аккаунта
  • Запрещены читы, инжекты и модификации, дающие преимущество вне движка.
  • Спорные ситуации решает администратор турнира.
Поле Значение
Турнир указать название
Режим duel / TDM / …
Сервер hostname, регион
Поддержка Discord / контакт

Сетевой профиль

Базовые значения при стабильной сети:

Параметр Значение
rate 25000
cl_maxpackets 125
cl_packetdup 0
cl_timeNudge 0
cl_autoTimeNudge 0

Подстройка под сеть

Ситуация Что сделать
Всё стабильно Базовый профиль выше
Потеря пакетов (cg_lagometer 1, зелёные spikes) cl_packetdup 1, при сильном loss - до 2-3
Ping до ~60 ms cl_timeNudge 0, cl_autoTimeNudge 0
Ping 70-100+ ms Опционально cl_timeNudge -10 … -20 (только отображение на экране)
cl_maxpackets 125 - стандарт; не снижать без явной причины

cl_timeNudge не меняет серверный lag compensation и факт попаданий.
